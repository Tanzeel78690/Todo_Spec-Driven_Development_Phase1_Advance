# Quickstart: Python CLI Todo App

This guide provides instructions on how to run the Python CLI Todo application.

## Prerequisites

-   Python 3.13 or higher.

## Running the Application

1.  Navigate to the root directory of the project.
2.  Run the application using the following command:

    ```bash
    python -m src
    ```

    (Assuming the main entry point is in `src/__main__.py`)

## Usage

Once the application is running, you will be presented with a menu of options. Simply type the number corresponding to the action you wish to perform and press Enter. Follow the on-screen prompts to manage your tasks.
